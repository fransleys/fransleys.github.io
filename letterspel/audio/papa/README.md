# Papa-opnames

De zestien bruikbare boodschappen uit `papa_raw.m4a`, verdeeld over `correct/` en `retry/`.

`index.json` bevat de bestandsnaam, uitgeschreven boodschap en een voorgesteld gewicht voor willekeurige selectie. Een hoger gewicht betekent dat de opname vaker mag worden afgespeeld. `papa-retry-07` is specifiek bedoeld voor telopdrachten.

Exportformaat: mono MP3, 48 kHz, 128 kbit/s. Iedere clip is genormaliseerd naar circa -16 LUFS met veilige piekbegrenzing.
