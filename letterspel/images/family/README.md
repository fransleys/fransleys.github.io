# Familiekaartjes

Eenentwintig naamloze portretten voor het letterspel. De zichtbare labels en hartjes uit de broncollage zijn verwijderd. Elk beeld is 354 × 414 pixels; de uiteindelijke flashcard-rand wordt door de interface getekend.

`index.json` koppelt ieder beeld aan de naam die de kinderen moeten vormen. Samengestelde namen hebben daarnaast een `words`-veld, zodat de interface ze over twee regels of in twee rondes kan aanbieden.
